# 📘 Calibre DRC/LVS Mastery

> A complete beginner-friendly guide to **Calibre nmDRC/nmLVS** — covering all 11 chapters of the *Calibre Using DRC/LVS* learning path. This repository includes detailed notes, slides, screenshots, and cheatsheets so anyone can understand Calibre without watching the videos.

---

## 🎯 What is This Repo?

This repository is a self-contained study resource for learning **Mentor Calibre**, one of the industry-leading EDA (Electronic Design Automation) tools used for:

- **DRC** — Design Rule Checking: verifying that your IC layout follows fabrication rules
- **LVS** — Layout vs. Schematic: ensuring your layout matches your schematic netlist

Whether you're a student, a fresh engineer, or someone switching into physical verification — this repo has everything you need.

## 🚀 How to Use This Repo

1. **Start with the notes** — Each chapter folder has a detailed `.md` file explaining every topic
